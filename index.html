WelCome to Attendance System .

This attendance system is basically developed for the offices in corporate and general world of business. I and my team-mate Md. Mushfiqur Rahman Shibly habe build this project.It is built with C# windows form and Microsoft SQL Server 2012. It will run in all windows platforms which supports .exe files.

Features

1.Give attendance with username and password which are supplied to each individual employee.
2.Admin can see the attendance history of each employer.
3.Holidays can be assigned separately.
4.Attendance history can be imported into excel sheet for printing or save for further use.
